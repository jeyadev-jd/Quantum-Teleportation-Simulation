## ğŸ“– About The Project

This project is a visually engaging and educational tool designed to demystify the process of quantum teleportation. It's a single, self-contained HTML file built with **Tailwind CSS** and vanilla **JavaScript** that runs entirely in the browser.

Users can select an initial quantum state for Alice, choose a noise model, and then watch step-by-step as the state is "teleported" to Bob. The interface provides real-time feedback, an animated quantum circuit, and explanations of the underlying physics.
